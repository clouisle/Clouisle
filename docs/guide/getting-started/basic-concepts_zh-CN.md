# 基础概念

有效使用 Clouisle 需要理解的核心概念。

## 团队与多租户

Clouisle 中的每个资源都属于一个团队。用户可以是多个团队的成员。

## AI Agent

可以使用工具、访问知识库和执行工作流的对话式 AI 助手。

## 知识库

具有向量搜索功能的文档存储库，用于 RAG（检索增强生成）。

## 工作流

具有 15+ 种节点类型的可视化自动化工作流，用于复杂的业务逻辑。

详细说明请参见[概念](../concepts/)。

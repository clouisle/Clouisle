# 登录与注册

本指南说明如何通过登录和注册访问 Clouisle。

## 注册方式

Clouisle 支持两种注册方式：

### 1. 自助注册

如果管理员启用了自助注册，您可以直接创建账号。

**步骤：**

1. 访问登录页面：`https://your-domain.com`
2. 点击**"注册"**或**"Sign Up"**
3. 填写注册表单：
   - **用户名**：唯一标识符（3-20 个字符）
   - **邮箱**：有效的电子邮件地址
   - **密码**：强密码（8+ 字符，包含字母、数字、符号）
   - **确认密码**：重新输入密码
4. 如果显示人机验证，请点击**"点击验证您是真人"**
5. 点击**"注册"**
6. 检查您的邮箱以获取验证链接（如果启用了邮箱验证）
7. 点击验证链接激活您的账号
8. 使用您的凭据登录

**注意**：如果需要管理员审批，您会看到账号待审批的消息。等待管理员激活您的账号。

### 2. 邀请注册

管理员可以通过电子邮件邀请用户。

**步骤：**

1. 收到管理员发送的邀请邮件
2. 点击邮件中的邀请链接
3. 填写您的密码和其他必需信息
4. 点击**"完成注册"**
5. 使用您的凭据登录

## 登录方式

### 密码登录

**步骤：**

1. 访问登录页面
2. 输入您的**用户名**或**邮箱**
3. 输入您的**密码**
4. 如果显示人机验证，请点击**"点击验证您是真人"**
5. （可选）勾选**"记住我"**以保持登录状态
6. 点击**"登录"**

**安全功能：**
- 多次登录失败后账号锁定
- CAPTCHA 验证（如果启用）
- 会话超时保护

### SSO（单点登录）

如果您的组织配置了 SSO，您可以使用企业凭据登录。

**步骤：**

1. 访问登录页面
2. 点击 SSO 提供商按钮（例如"使用 Google 登录"、"使用 GitHub 登录"）
3. 使用您的 SSO 提供商进行身份验证
4. 您将被重定向回 Clouisle 并自动登录

**首次 SSO 登录：**
- 如果您的邮箱匹配现有账号，SSO 连接将被关联
- 如果不存在账号，将自动创建新账号（如果启用）

**支持的 SSO 提供商：**
- OAuth2/OIDC（Google、GitHub、GitLab 等）
- SAML 2.0（Azure AD、Okta、OneLogin）
- CAS（大学系统）

有关 SSO 配置，请参阅 [SSO 用户指南](./sso-user-guide_zh-CN.md)。

## 首次登录

首次成功登录后：

1. **欢迎屏幕**：您将看到欢迎消息
2. **个人资料设置**：完善您的个人资料信息（可选）
3. **团队分配**：
   - 如果被邀请加入团队，您将在团队列表中看到它
   - 如果未分配团队，您可能需要创建一个或等待邀请
4. **仪表板访问**：
   - 普通用户看到平台界面
   - 管理员看到管理仪表板

## 会话管理

### 会话时长

- **默认**：30 分钟不活动
- **记住我**：延长会话（由管理员配置）
- **单会话模式**：每个用户只能有一个活动会话（如果启用）

### 保持登录

**"记住我"选项：**
- 让您保持较长时间的登录状态
- 适用于个人设备
- **不要在共享计算机上使用**

### 登出

**手动登出：**
1. 点击右上角的个人资料图标
2. 选择**"登出"**
3. 您将被重定向到登录页面

**自动登出：**
- 会话超时后（不活动）
- 管理员停用您的账号时
- 从另一台设备登录时（如果启用单会话模式）

## 账号安全

### 密码要求

Clouisle 强制执行强密码策略：

- **最小长度**：8 个字符（可配置）
- **复杂度**：大小写字母、数字和符号的混合
- **无常见密码**：拒绝字典单词和常见模式
- **不重复使用**：不能重复使用最近的密码（如果启用）

### 登录安全功能

**账号锁定：**
- 5 次登录失败后（可配置）
- 账号锁定 15 分钟
- 如果反复被锁定，请联系管理员

**人机验证：**
- 管理员启用后，登录和自助注册都需要完成验证
- 提交前点击验证控件即可完成
- 如果验证失败、超时或加载异常，请点击刷新按钮后重新验证
- 与账号锁定和登录异常检测一起作为风险控制兜底

**登录异常检测：**
- 系统跟踪您的常用登录位置和设备
- 从新位置登录时通知您
- 如果看到异常警报，请检查通知

### 双因素认证（2FA）

如果您的组织启用了 2FA：

1. 输入用户名和密码
2. 输入验证码，来自：
   - 身份验证器应用（Google Authenticator、Authy）
   - 短信（如果配置）
   - 电子邮件（如果配置）
3. 点击**"验证"**

**设置 2FA：**
1. 转到**个人资料设置** → **安全**
2. 点击**"启用双因素认证"**
3. 使用身份验证器应用扫描二维码
4. 输入验证码确认
5. 将备份代码保存在安全的地方

## 故障排查

### 无法访问登录页面

**问题**：登录页面无法加载

**解决方案：**
1. 检查您的互联网连接
2. 验证 URL 是否正确
3. 尝试不同的浏览器
4. 清除浏览器缓存和 Cookie
5. 联系您的 IT 管理员

### 忘记密码

**问题**：记不住密码

**解决方案：**
1. 在登录页面点击**"忘记密码？"**
2. 输入您的电子邮件地址
3. 检查您的邮箱以获取重置链接
4. 点击链接并输入新密码
5. 使用新密码登录

详见[密码管理](./password-management_zh-CN.md)。

### 账号被锁定

**问题**：登录失败多次后显示"账号已锁定"

**解决方案：**
1. 等待 15 分钟自动解锁
2. 联系管理员立即解锁
3. 如果忘记密码，使用密码重置

### 未收到验证邮件

**问题**：没有收到验证邮件

**解决方案：**
1. 检查垃圾邮件/垃圾文件夹
2. 等待几分钟（邮件可能延迟）
3. 在登录页面点击**"重新发送验证邮件"**
4. 验证电子邮件地址是否正确
5. 如果仍未收到，请联系管理员

### SSO 登录失败

**问题**：使用 SSO 登录时出错

**解决方案：**
1. 验证您使用的是正确的 SSO 提供商
2. 检查您的 SSO 账号是否处于活动状态
3. 清除浏览器 Cookie 并重试
4. 联系您的 SSO 管理员
5. 如果可用，尝试密码登录

详见 [SSO 用户指南](./sso-user-guide_zh-CN.md) 了解 SSO 特定的故障排查。

### "账号待审批"消息

**问题**：无法登录，账号待审批

**解决方案：**
- 您的账号需要管理员审批
- 等待管理员激活您的账号
- 批准后您将收到电子邮件
- 如果等待时间过长，请联系管理员

### 会话过期

**问题**：使用过程中显示"会话已过期"

**解决方案：**
1. 重新登录
2. 启用"记住我"以延长会话
3. 联系管理员调整会话超时时间
4. 检查是否在另一台设备上登录（单会话模式）

## 最佳实践

### 安全建议

**✅ 应该：**
- 使用强且唯一的密码
- 启用双因素认证
- 在共享计算机上使用后登出
- 保持电子邮件地址最新
- 定期查看登录通知
- 如果可用，使用 SSO（更安全）

**❌ 不应该：**
- 与他人共享密码
- 使用与其他服务相同的密码
- 在公共计算机上保持登录状态
- 忽略登录异常通知
- 使用简单或常见的密码

### 账号管理

**定期任务：**
- 每 90 天更新密码（如果需要）
- 定期查看活动会话
- 检查登录历史以发现可疑活动
- 保持个人资料信息最新
- 更新通知偏好设置

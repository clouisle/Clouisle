## [0.1.0] - 2025-02-07

Clouisle 首个公开发布版本 - 企业级知识库与 AI Agent 平台。

### 核心功能

#### Agent 系统

- **Agent 类型**：支持聊天 Agent 和工作流 Agent
- **Agent 编排**：现代化三栏布局，实时预览
  - 提示词编辑器，支持 AI 生成
  - 变量配置
  - 知识库关联
  - 工具配置
  - 视觉能力支持
- **Agent 发布**：发布/取消发布工作流，状态管理
- **Agent API 访问**：OpenAI 兼容 API，提供文档和代码示例
- **Agent 监控**：使用统计、对话趋势、Token 用量、响应时间指标

#### 工作流引擎

- **可视化工作流编辑器**：基于 React Flow 的拖拽式工作流构建器
- **节点类型**：
  - 开始/结束节点
  - LLM 节点（支持模型选择和参数调优）
  - 条件节点（分支逻辑）
  - 代码节点（JavaScript/Python 执行）
  - 工具节点（HTTP API、MCP、内置工具）
  - 迭代/循环节点
  - 变量赋值/聚合节点
  - 子工作流节点
  - 问题分类器节点
  - 参数提取器节点
- **变量系统**：上游变量引用与校验
- **执行**：实时流式执行，节点级状态跟踪

#### 知识库

- **文档管理**：
  - 多格式支持：PDF、DOCX、TXT、MD、HTML、CSV、XLSX、JSON、PPTX
  - URL 导入
  - 拖拽批量上传
- **文档处理**：
  - 智能文本分块（可配置 chunk_size、chunk_overlap、separator）
  - 提交前分块预览
  - 文本清洗选项
- **搜索模式**：
  - 向量搜索（语义相似度）
  - 全文搜索（jieba 中文分词）
  - 混合搜索（RRF 算法）
- **动态 Embedding**：支持多种 embedding 维度（768、1024、1536、3072）

#### 聊天系统

- **通用聊天组件**：可复用的聊天组件库
- **消息类型**：文本、推理（思维链）、工具调用、文件、来源
- **流式输出**：实时流式响应，Streamdown 渲染
- **多轮对话**：对话历史管理

#### 工具系统

- **内置工具**：计算器、时间/时区
- **HTTP API 工具**：可配置 URL、方法、Headers、Body 模板
- **MCP 服务器工具**：stdio/sse 传输、命令执行
- **代码工具**：在线 JavaScript/Python 编辑器

### 平台功能

#### 用户管理

- **认证**：用户名/密码、SSO（OIDC、SAML、CAS）
- **注册**：可配置注册、审批流程、邮箱验证
- **密码策略**：可配置强度要求
- **会话管理**：超时、单一会话模式
- **登录安全**：最大尝试次数、锁定、验证码

#### 团队管理

- **多租户**：基于团队的资源隔离
- **基于角色的访问控制**：可自定义角色和权限
- **团队成员**：邀请、移除、角色分配

#### 模型管理

- **多供应商支持**：OpenAI、Anthropic、Azure、本地模型
- **模型授权**：团队级模型访问控制
- **用量跟踪**：按团队统计 Token 用量

#### API 密钥管理

- **API 密钥 CRUD**：创建、轮换、过期 API 密钥
- **Agent 绑定**：限制 API 密钥访问特定 Agent
- **速率限制**：可配置速率限制

#### 站点设置

- **常规**：站点名称、描述、URL、图标
- **默认语言**：可配置系统消息的默认语言
- **安全**：注册、密码策略、会话、登录安全
- **通知**：邮件（SMTP）、钉钉、企业微信、飞书、Slack、Webhook
- **自动通知**：可配置系统事件通知
- **SSO**：OIDC、SAML、CAS 提供商配置
- **存储**：审计日志保留和归档

#### 审计日志

- **全面记录**：记录所有用户操作
- **日志查看器**：可筛选、可搜索的审计日志界面
- **导出**：支持 CSV 导出
- **归档**：可配置保留期限，自动归档

### 国际化

- **语言**：英语、简体中文
- **用户语言**：每用户语言偏好
- **站点默认**：可配置系统消息的默认语言
- **动态 i18n**：所有用户界面消息国际化

### UI/UX

- **主题**：明/暗模式，支持系统偏好检测
- **动态 Favicon**：主题感知的 Favicon 切换
- **响应式设计**：桌面端优化的仪表板
- **组件库**：基于 shadcn/ui 的组件

### 技术栈

#### 后端
- Python 3.13、FastAPI
- Tortoise ORM + AsyncPG
- Celery 后台任务
- Redis 缓存和发布/订阅
- Qdrant 向量存储

#### 前端
- Next.js 15、React 19
- TypeScript
- Tailwind CSS
- shadcn/ui 组件
- Bun 包管理器

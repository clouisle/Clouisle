# 工作流设计模式

常见的工作流设计模式。

## 顺序模式

线性执行: A → B → C → D

## 并行模式

并发执行: A → (B, C, D) → E

## 条件模式

分支逻辑: A → if(条件) → B else C

## 循环模式

迭代执行: A → while(条件) → B → A

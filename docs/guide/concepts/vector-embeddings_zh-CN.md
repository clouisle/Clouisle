# 向量嵌入

理解向量嵌入和相似度搜索。

## 什么是嵌入？

嵌入是捕获语义含义的文本数值表示。

## 相似度搜索的工作原理

1. 将查询转换为向量
2. 在数据库中搜索相似向量
3. 返回最相似的文档

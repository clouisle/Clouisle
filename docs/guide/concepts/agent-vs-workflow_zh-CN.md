# Agent vs Workflow

理解 Agent 和 Workflow 之间的区别。

## Agent

**对话式和自主**:
- 交互式聊天界面
- 多轮对话
- 自主决策

## Workflow

**结构化和确定性**:
- 可视化节点编辑器
- 预定义执行路径
- 显式控制流

详细对比请参考英文版本。
